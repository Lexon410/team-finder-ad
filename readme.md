# TeamFinder —  (вариант 3)

Проект представляет собой веб-платформу для поиска единомышленников для pet-проектов.  
В данном варианте реализована функциональность **навыков проектов** и фильтрация проектов по навыкам.

## Запуск проекта (локальная разработка)

### 1. Клонирование репозитория

```bash

cd team-finder-ad
python -m venv venv
# для Windows:
venv\Scripts\activate
# для Linux/macOS:
source venv/bin/activate


pip install -r requirements.txt


cp .env_example .env


docker compose up -d

python manage.py migrate
python manage.py createsuperuser

python manage.py runserver
